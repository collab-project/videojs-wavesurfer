    private formatTime(seconds: number, guide: number, msDisplayMax: number): string source 

    import formatTime from 'videojs-wavesurfer/src/js/utils/format-time.js'

    Format seconds as a time string, H:MM:SS, M:SS or M:SS:MMM.

    Supplying a guide (in seconds) will force a number of leading zeros to cover the length of the guide.

    Params:

    NameTypeAttributeDescription
    seconds number

    Number of seconds to be turned into a string.
    guide number

    Number (in seconds) to model the string after.
    msDisplayMax number

    Number (in milliseconds) to model the string after.

    Return:

    string

    Time formatted as H:MM:SS, M:SS or M:SS:MMM, e.g. 0:00:12.
